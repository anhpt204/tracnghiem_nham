The adapter to use MATSim as a travel model plug-in to UrbanSim.

The current travel model plug-in implementation is applicable for PSRC (Puget Sound Region Council) parcel, Seattle parcel and Zurich parcel scenario. It was also used for a Brussels zone scenario. Other scenarios may work or not.

This module has not been maintained since 2013 and it is unclear how much of it still works.

For more information, see here and the javadoc.
